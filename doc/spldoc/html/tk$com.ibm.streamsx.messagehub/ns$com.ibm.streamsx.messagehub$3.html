Tracing Kafka performance metrics

Message Hub Toolkit > com.ibm.streamsx.messagehub 1.8.0 > com.ibm.streamsx.messagehub > Tracing Kafka performance metrics

For testing and tuning it might be desirable to see and observe performance metrics of the Kafka consumer or Kafka Producer client. When the application trace level is at INFO (or at a more verbose level), the operators write every two seconds the most important performance metrics into the trace file. These metrics can be used to tune the Kafka consumer or producer by applying specific Kafka configs.
