Tracing Kafka properties

Message Hub Toolkit > com.ibm.streamsx.messagehub 2.1.0 > com.ibm.streamsx.messagehub > Tracing Kafka properties

For testing it might desirable to see all effective Kafka Consumer or Producer properties, also those properties, which have not been setup by application configuration or configuration file. To achieve this, submit the Streams application with application trace level INFO or more verbose. Then the properties are traced into the PE trace. Authentication information is hidden.
